## 📋 Présentation
**insta-manager-pro** est le moteur principal de ton système SMM. Il contient toute la logique de gestion des comptes, de connexion et d'automatisation.

> [!IMPORTANT]
> Ce projet est conçu pour être lancé via l'installeur sécurisé. Ne tentez pas de le lancer manuellement sans activation préalable.

## 🚀 Fonctionnalités incluses
D'après ton interface de contrôle, cet outil permet de :
* **Démarrer le Bot** : Lancement des scripts d'automatisation.
* **Gestion des comptes** : Connexion, listage et suppression de comptes.
* **Sécurité HWID** : Protection par clé unique par appareil.
* **Mises à jour** : Système intégré de mise à jour automatique.

## ⚙️ Structure du projet
Le dossier contient les éléments suivants :
* `update.py` : Script de mise à jour et lancement.
* `sessions/` : Dossier sécurisé stockant les comptes connectés.
* `core/` : Logique interne du bot.
